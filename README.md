# Simple Cameras

A Fabric mod for Minecraft 1.21.1 that adds security cameras and monitoring systems.

## Features

- **Security Camera Block**: Place cameras that face different directions
- **Camera Monitor Block**: Right-click to view all nearby cameras within range

## Building

```bash
./gradlew build
```

The mod JAR will be generated in `build/libs/`

## Installation

1. Download the latest JAR from releases
2. Place it in your `.minecraft/mods/` folder
3. Make sure you have Fabric installed

## Requirements

- Minecraft 1.21.1
- Fabric Loader 0.16.0+
- Fabric API
- Java 21+
